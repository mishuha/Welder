# Welder
Arduino transformer welding controller with 2 impulses mode.  
  
Configurable:  
1. preheat cycle time  
2. pause  
3. main cycle time  

All cycles start on feedback impulse. SSR is supposed to commutate load after crossing zero level of sin wave.
